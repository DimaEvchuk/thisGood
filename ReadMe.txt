Дизайн пока сыроват, но есть что показать
Проект на платформе ASP.NET Core с применение паттерна MVC
В проекте реализованно:
1) Использование паттера 'Репозиторий' с Entity Framework (EF) для работы с БД
2) локальная БД на MS SQL Server
3) запросы к БД с помощью библиотеки LINQ
4) использование паттерна MVVM
5) корзина с помощью Razor Pages
6) страница Admin
